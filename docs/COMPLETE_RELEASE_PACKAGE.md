## 📋 发布步骤

### 第 1 步：选择图标方案
- 查看 `docs/ICON_DESIGNS.md`
- 选择喜欢的方案 (1-5)
- 导出为 512x512 PNG

### 第 2 步：准备截图
- 参考 `docs/SCREENSHOT_GUIDE.md`
- 获取 5-8 张截图
- 添加配文 (可选)

### 第 3 步：部署隐私政策
- 上传 `docs/privacy_policy.html` 到 GitHub Pages
- 或部署到其他服务器
- 获取公开 URL

### 第 4 步：注册 Google Play
- 访问：https://play.google.com/console/signup
- 支付 $25 注册费
- 完成身份验证

### 第 5 步：创建应用
- 登录 Google Play Console
- 创建新应用
- 填写应用信息
- 上传图标和截图

### 第 6 步：提交审核
- 检查所有项目
- 上传 signed APK
- 提交审核
- 等待 1-3 天

### 第 7 步：发布上线
- 审核通过
- 自动发布
- 🎉 上线成功！
