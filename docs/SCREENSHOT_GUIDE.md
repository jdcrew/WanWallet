## 📸 截图获取方式

### 方式 1: Android Studio 模拟器

```bash
# 1. 在 Android Studio 中运行应用
# 2. 使用模拟器截图按钮
# 3. 或使用快捷键:
#    - Windows/Linux: Ctrl + S
#    - Mac: Cmd + S
```

### 方式 2: ADB 命令

```bash
# 连接设备后执行
adb shell screencap -p /sdcard/screenshot1.png
adb pull /sdcard/screenshot1.png ./screenshots/home.png

# 重复获取所有截图
```

### 方式 3: 真机截图

```bash
# 在真机上使用音量下 + 电源键截图
# 然后通过 USB 传输到电脑
```

## 🎨 截图美化建议

### 添加设备外壳

**工具:**
- https://mockuphone.com/ (免费)
- https://artboard.studio/ (付费)

**步骤:**
1. 截取应用页面
2. 上传到 Mockuphone
3. 选择手机型号
4. 下载带外壳的截图

### 添加配文

**工具:**
- Canva
- Figma
- Photoshop

**建议:**
- 字体：思源黑体 / Noto Sans
- 字号：24-32px
- 颜色：白色 (深色背景) / 黑色 (浅色背景)
- 位置：截图下方或上方

## 📐 技术规格

### Google Play 要求

| 项目 | 要求 |
|------|------|
| 最小宽度 | 320px |
| 最大宽度 | 3840px |
| 宽高比 | 16:9 或 9:16 |
| 格式 | PNG 或 JPG |
| 数量 | 最少 2 张，最多 8 张 |

### 推荐规格

| 项目 | 推荐值 |
|------|--------|
| 尺寸 | 1080x1920 (竖屏) |
| 格式 | PNG |
| 数量 | 5-6 张 |
