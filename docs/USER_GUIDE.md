# WanWallet 用户指南

## 快速开始

### 1. 首次设置

1. **打开应用** - 启动 WanWallet
2. **授予权限** - 开启通知监听权限
   - 点击"去开启"跳转到设置页面
   - 找到 WanWallet 并开启通知监听
   - 返回应用继续设置
3. **可选：无障碍服务** - 提高识别准确率
   - 点击"去开启"跳转到无障碍设置
   - 找到 WanWallet 并开启

### 2. 自动记账

**无需手动操作！**

WanWallet 会自动：
- 监听微信/支付宝/云闪付支付通知
- 提取金额和商户信息
- 自动分类 (基于规则引擎)
- 保存到交易记录

### 3. 手动记账

如需手动添加：
1. 点击底部导航栏"+"按钮
2. 输入金额
3. 选择支出/收入类型
4. 选择分类 (会自动匹配)
5. 输入商户名称
6. 选择支付渠道
7. 添加备注 (可选)
8. 点击"保存"

### 4. 查看统计

**底部导航栏：**
- 🏠 **首页** - 月度概览、快速操作、最近交易
- 📊 **统计** - 收支统计、分类占比、趋势图表
- ⚙️ **设置** - 权限管理、数据导出、关于

### 5. 预算管理

1. 进入"统计"页面
2. 点击"预算"标签
3. 点击"+"添加预算
4. 设置预算金额和周期
5. 查看预算进度和超支警告

### 6. 数据导出

1. 进入"设置"页面
2. 点击"导出数据"
3. 选择 CSV 或 Excel 格式
4. 选择时间范围
5. 导出文件保存到手机

## 常见问题

### Q: 为什么没有自动记录交易？

**A:** 请检查：
1. 通知监听权限是否开启
2. 微信/支付宝通知是否开启
3. 后台运行权限是否允许

### Q: 分类不准确怎么办？

**A:** 可以手动修改分类，系统会学习您的选择。

### Q: 数据会上传到云端吗？

**A:** 不会！所有数据存储在本地，保护您的隐私。

### Q: 如何备份数据？

**A:** 使用"数据导出"功能，导出 CSV/Excel 文件保存。

### Q: 支持哪些支付平台？

**A:** 目前支持：
- 微信支付
- 支付宝
- 云闪付
- 现金
- 银行卡

## 隐私政策

**数据本地存储** - 所有交易数据存储在您的设备本地

**无云端同步** - 不会上传任何数据到服务器

**权限说明：**
- 通知监听：用于捕获支付通知
- 无障碍服务：用于提高识别准确率 (可选)
- 网络：用于 LLM 分类 (可选，可关闭)
